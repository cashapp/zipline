interface SuspendCallback<T> : ZiplineService

A bridged interface to return results from suspending calls. This is used internally by Zipline to complete suspending calls from the receiving endpoint to the calling endpoint.

It is not necessary for the receiving endpoint to ZiplineService.close this; that's handled automatically by the calling service.
